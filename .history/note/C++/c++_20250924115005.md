# C++

## C++基本语法

```c++
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
    return 0;
}
```

## 基本数据类型

- 整型：`int`、`short`、`long`、`long long`
- 浮点型：`float`、`double`
- 字符串型：`string`
- 宽字符型：`wchar_t`  (`typedef short int wchar_t`)
- 字符型：`char`
- 布尔型：`bool`
- 空类型：`void`

## 类型修饰符

- `signed`：有符号
- `unsigned`：无符号
- `short`：短整型
- `long`：长整型
- `const`：常量
- `volatile`：易失性
- `extern`：外部变量
- `auto`：自动
- `register`：寄存器
- `static`：静态

## c++新增类型

- `nullptr`：空指针
- `auto`：自动类型推导
- `decltype`：类型推导
- `decltype(auto)`：类型推导
- `constexpr`：常量表达式
- `noexcept`：异常说明
- `override`：重写
