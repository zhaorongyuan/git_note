# C++

## C++基本语法

```c++
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
    return 0;
}
```

## 基本数据类型

- 整型：`int`、`short`、`long`、`long long`
- 浮点型：`float`、`double`
- 宽字符型：`wchar_t`  (`typedef short int wchar_t`)
- 字符型：`char`
- 布尔型：`bool`
- 空类型：`void`
- 字符串型：`string`
